# site-toninho

## Description

A standard software project.

## Installation

1.  Clone the repository: `git clone https://github.com/site-toninho/site-toninho.git`
2.  Navigate to the project directory: `cd site-toninho`
3.  Install dependencies: `npm install` (or equivalent for your package manager)
4.  Run the development server: `npm start`

## Usage

*   **View the Site:**  Open `index.html` in your web browser.
*   **Customize:**
    *   Modify the `_headers` file to adjust your site's HTML structure.
    *   Adjust the `_redirects` file to configure redirects.
    *   Edit the `custom.css` file to personalize the styling.
    *   Modify the `fotoAntonio.jpg` image.
    *   Use the `SEO_KEYWORDS.md` file for keyword optimization.
    *   Update the `script.js` file for interactivity.
    *   Review the `sitemap.xml` file for SEO purposes.
    *   Ensure `video.mp4` is properly configured (e.g., using the appropriate media player).
*   **Build & Deploy:**  Run `npm run build` to generate static assets.  Then, deploy to Netlify using `netlify.toml` (refer to documentation).
